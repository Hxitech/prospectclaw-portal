Step by Step

一步一步来，不跳步

8 个步骤，按顺序来。每一步做完确认"过关标准"再往下走。

01
你需要准备什么

只需要一台能上网的电脑

  • 一台电脑（Mac 或 Windows），需要能联网。手机平板不行。
  • 不需要编程经验，不需要懂代码，不需要上过计算机课。
  • 不需要知道"命令行"是什么。如果后面用到，我们会明确告诉你输入什么。
  • 有技术同事更好，但不是必须——你有 AI 助手可以问。
过关标准： 你有一台能上网的 Mac 或 Windows 电脑。这就够了。
02
下载和安装

像装普通软件一样安装

去主站下载安装包：点击这里去下载页面

Mac 用户：

  • 下载 .dmg 文件，双击打开，把图标拖到"应用程序"文件夹。
  • 重要：Mac 可能提示"无法验证开发者"或"已损坏"。这不是病毒！解决方法：
    点击左上角苹果图标 → 系统设置 → 隐私与安全性 → 找到 ProspectClaw 被阻止的提示 → 仍要打开

Windows 用户：

  • 下载 .exe 安装包，双击运行，一路"下一步"即可。
  • 如果出现安全提示，选择"仍要运行"。
过关标准： ProspectClaw 已安装，你可以双击打开它了。
03
第一次打开 — 初始设置

让应用帮你检查电脑环境

第一次打开会自动进入"初始设置"页面，帮你检查需要的工具是否齐全。

  • 绿色勾 = 已通过，不用管。
  • 红色叉 = 需要处理，按下面说明来。

如果 Node.js 显示红色：

  • Node.js 是一个运行环境工具（不需要理解它，装上就行）。
  • 打开浏览器去 nodejs.org，下载安装即可。
  • 装完后回到 ProspectClaw 刷新检查。

如果 OpenClaw 显示红色：

  • 直接点页面上的"安装"按钮，等它自动完成（几分钟）。

什么是"命令行"？

一个可以输入文字指令的窗口。Mac：按 Command+空格，搜 Terminal 回车。Windows：按 Win 键，搜 cmd 回车。正常情况下初始设置的按钮就够用了。

过关标准： 初始设置页面上所有检查项都是绿色的。有红色项搞不定？先往下走，配好钳子助手（第 4 步）后让它帮你排查。
04
配置钳子助手（AI 私人教练）

先叫醒你的"私人教练"

在左侧菜单找到"钳子助手"，点进去。它是内置的 AI 助手，能帮你解答问题、排查故障。

但它现在还不能工作——需要给它配一个"AI 大脑"（模型）。

  • 点右上角"设置"按钮。
  • 填写 AI 模型信息，需要一个 API Key（使用密码）。
  • 不知道用什么模型？试试 MiniMax（有免费额度），或联系技术同事拿一个 API Key。
  • 填好后点"测试连接"，看到成功提示就配好了。
过关标准： 钳子助手能正常对话。试试输入"你好"看它是否回复。从现在开始，遇到问题直接问它！
从这一步开始，你有帮手了

后面遇到任何问题，都可以在钳子助手里描述你的问题。你也可以把这个网页链接发给它。再也不用一个人硬扛了。

05
配置主引擎（模型和 Gateway）

给系统装上"发动机"

第 4 步配的是钳子助手自己的模型。现在给系统主引擎也配上。

  • 左侧菜单找到"模型配置"，添加至少一个 AI 模型（可以和钳子助手用同一个）。
  • 点"测试"确认模型能用。
  • 然后去"仪表盘"或"服务管理"页面。
  • 找到 Gateway（系统总开关），点"启动"。
  • 等它变绿色"运行中"，主引擎就启动成功了。
过关标准： 模型测试通过，Gateway 显示"运行中"（绿色）。Gateway 启动不了？问钳子助手。
06
第一次对话 — 验证系统

发一条消息，验证系统跑通了

  • 左侧菜单找到"实时聊天"，点进去。
  • 输入："你好，介绍一下你自己"，按回车。
  • 收到回复 = 系统已完全跑通！
  • 没收到回复？检查 Gateway 是否还在"运行中"，或问钳子助手："实时聊天发消息没回复，怎么办？"
过关标准： 实时聊天里发消息能收到回复。核心功能已跑通。
07
探索更多功能（可选）

系统跑通后，这些功能值得试试

基础链路已通，可以按兴趣探索这些模块：

AI 能力：

  • SkillHub 技能商店（v1.2.8 新增）— 在线浏览、搜索、一键安装 Agent 技能。去"Skills"页面的"商店"标签页体验。
  • Prospect-Research 科学家集群 — 多 AI 协作完成复杂任务，自带 12 个案例模板。
  • ClawSwarm 蜂群协作 — 输入目标，AI 自动拆解为任务 DAG，多智能体并行执行。

投资管理（Invest 版本）：

  • 投资概览 — 项目池漏斗、管道分布和近期动态仪表盘。
  • Deal Pipeline — 六阶段看板管理（寻源→初筛→尽调→IC→条款→关闭）。
  • 企业库 + 联系人 — 被投企业档案和联系人管理。

消息渠道（可选）：

  • 飞书 / Lark — 需要在飞书开放平台创建应用，拿到 App ID 和 App Secret。操作较复杂，建议让钳子助手或技术同事协助。
  • Telegram — 通过 @BotFather 创建 Bot 拿到 Token 即可，最简单。
过关标准： 你已经知道系统里有哪些功能，并按兴趣试了几个。
08
遇到问题怎么办

三个求助方法，优先级排序

  • 第一选择：问钳子助手！描述问题，比如"Gateway 还是红色的"，它会帮你排查。
  • 第二选择：发给外部 AI。把这个页面链接发给 ChatGPT / Claude，告诉它你的问题。
  • 第三选择：查看进阶指南。进阶指南有更详细的操作手册和排障清单。

最重要：不要一出问题就重装！大多数问题有简单解决方法，重装反而可能丢配置。先问 AI。

过关标准： 你知道遇到问题该找谁——先找钳子助手，或把这个页面发给 AI。
Quick FAQ

最常见的 4 个问题

🍎

Mac 提示"无法打开"或"已损坏"

不是病毒，是苹果安全机制。系统设置 → 隐私与安全性 → 找到 ProspectClaw → 仍要打开。详见第 2 步。

🔴

初始设置里有红色项

红色项表示需要安装某工具。按页面提示操作。搞不定就问钳子助手："初始设置里 XX 是红色的，怎么办？"

🚧

Gateway 启动不了

最常见原因是模型没配好或端口被占用。直接在钳子助手里说"Gateway 启动失败"，让它帮你诊断。

💬

实时聊天没收到回复

先检查服务管理页面 Gateway 是否绿色"运行中"。不是就先启动它。是绿色还没回复，问钳子助手帮你查。

万能求助法

任何时候遇到问题，把这个页面链接（或截图）发给 AI 助手。无论是钳子助手、ChatGPT 还是 Claude，它们都能根据这里的信息帮你解决。
