零基础新手指南

从零开始，手把手带你用起来

这份指南专门为完全没有技术背景的用户准备。你不需要懂编程，不需要懂命令行，甚至不需要知道什么是"终端"。
只要照着下面的步骤一步一步做，就能把 ProspectClaw 用起来。
遇到任何问题，随时可以把这个页面的链接发给 AI 助手（钳子助手、ChatGPT、Claude 等），它们能根据这里的内容帮你排查。

从第一步开始 我遇到问题了
需要什么基础
不需要任何技术基础。会用电脑上网、会安装普通软件就够了。完全零基础也没关系，我们每一步都会详细说明。
大概需要多久
整个过程大约 30 分钟。如果中间需要下载安装一些工具，可能会久一点，取决于网速。不着急，慢慢来。
遇到问题怎么办
把这个页面的链接发给任何 AI 助手，告诉它你卡在哪一步、看到了什么提示，它就能帮你解决。你不需要自己懂技术。
Step by Step

一步一步来，不跳步

下面一共 8 个步骤。请按顺序来，每一步做完确认"过关标准"再往下走。

01
你需要准备什么

只需要一台能上网的电脑

在开始之前，先确认你手边有这些东西。放心，要求非常简单：

  • 一台电脑（Mac 或 Windows 都可以），需要能连上网。手机和平板不行，必须是电脑。
  • 不需要编程经验，不需要懂代码，不需要上过计算机课。
  • 不需要知道"命令行"是什么。如果后面某一步需要用到它，我们会在那个时候告诉你怎么打开、输入什么、怎么关闭。现在完全不用管它。
  • 如果你有一个懂技术的同事或朋友，遇到困难时可以找他们帮忙，但这不是必须的 —— 因为你还有 AI 助手可以问。
过关标准： 你有一台能上网的电脑（Mac 或 Windows）。这就够了，可以往下走了。
02
下载和安装

像装普通软件一样，把 ProspectClaw 装到电脑上

首先，去主站下载 ProspectClaw 安装包：点击这里去下载页面

如果你用的是 Mac（苹果电脑）：

  • 下载后你会得到一个 .dmg 文件，双击打开它。
  • 你会看到一个窗口，里面有 ProspectClaw 的图标和一个"应用程序"文件夹。把图标拖到文件夹上就行了。这跟你装其他 Mac 软件是一模一样的。
  • 重要提醒：Mac 很可能会弹出一个提示说"无法验证开发者"或者"已损坏，无法打开"。这不是病毒！这只是因为苹果没有认证这个软件。解决方法很简单：
    1. 点击屏幕左上角的苹果图标 → 选择"系统设置"
    2. 在左侧找到"隐私与安全性"，点进去
    3. 往下滚动，你会看到关于 ProspectClaw 被阻止的提示
    4. 点击"仍要打开"按钮
    5. 再次打开 ProspectClaw，这次就能正常启动了

如果你用的是 Windows：

  • 下载后你会得到一个安装包（.exe 文件），双击运行它。
  • 按照安装向导的提示，一路点"下一步"就行了。
  • 如果 Windows 弹出安全提示，选择"仍要运行"即可。
过关标准： ProspectClaw 已经安装到电脑上，你可以双击打开它了。
03
第一次打开 — 初始设置

让应用帮你检查电脑环境

第一次打开 ProspectClaw 后，它会自动进入一个叫"初始设置"的页面。这个页面会帮你检查电脑上是否已经准备好了需要的工具。

你会看到几个检查项，每个旁边会显示一个状态：

  • 绿色勾 = 这项已经通过，不用管。
  • 红色叉 = 这项需要处理。别紧张，按下面的说明来就行。

如果 Node.js 显示红色：

  • Node.js 是一个运行环境工具（你不需要理解它是什么，只需要装上它）。
  • 打开浏览器，去 nodejs.org 这个网站。
  • 你会看到一个绿色的下载按钮，点击下载，然后像装普通软件一样安装。
  • 安装完成后，回到 ProspectClaw，刷新检查，它应该就变绿色了。

如果 OpenClaw 显示红色：

  • 直接点击页面上的"安装"按钮，等它自动完成就好了。这个过程可能需要几分钟，耐心等待。

什么是"命令行"？

你可能会在一些提示里看到"命令行"这个词。命令行就是电脑上一个可以输入文字指令的窗口，长得像一个黑色（或白色）的文字框。在这个指南里，如果需要用到命令行，我们会明确告诉你要输入什么。

  • Mac 上怎么打开命令行：同时按下 Command 键和空格键，会弹出一个搜索框，在里面输入 Terminal（终端），然后按回车键。
  • Windows 上怎么打开命令行：按键盘上的 Win 键（左下角有 Windows 标志的那个键），在弹出的搜索框里输入 cmd，然后按回车键。
  • 不过正常情况下，初始设置页面上的按钮就够用了，你可能根本不需要自己打开命令行。
过关标准： 初始设置页面上所有检查项都是绿色的。如果有红色项怎么也搞不定，先别着急，等配好钳子助手（第 4 步）之后可以让它帮你排查。
04
配置你的 AI 助手（钳子助手）

先叫醒你的"私人教练"

在软件左侧的菜单栏里，找到"钳子助手"，点进去。

钳子助手是这个软件里内置的 AI 助手，你可以把它理解成你的"私人教练"。它能帮你解答问题、排查故障、告诉你下一步该做什么。

但是，它现在还不能工作，因为还没有给它配一个"AI 大脑"（也就是 AI 模型）。就好比你买了一台电视，但还没接上信号源，所以还看不了节目。

  • 点击右上角的"设置"按钮。
  • 你需要填写一个 AI 模型的信息。这里需要一个叫 API Key 的东西（可以理解为"使用密码"）。
  • 如果你不知道用什么模型：可以先试试 MiniMax（国产模型，有免费额度），或者联系你的技术同事帮你拿一个 API Key。
  • 填好后点"测试连接"，如果看到成功的提示，就说明配好了。
过关标准： 钳子助手能正常对话了。你可以试着输入"你好"看看它是否回复。从现在开始，遇到任何问题你都可以直接问它！
从这一步开始，你有帮手了

从这一步开始，如果后面遇到任何问题，你都可以直接在钳子助手里描述你的问题，它会帮你分析和解决。你也可以把这个网页的链接发给它，让它读取里面的信息来帮你。再也不用一个人硬扛了。

05
配置主引擎（模型和 Gateway）

给系统装上"发动机"

刚才第 4 步配的是钳子助手自己的模型。现在要给系统的主引擎也配上模型，这样整个系统才能正常工作。

  • 在左侧菜单找到"模型配置"，点进去。
  • 这里要添加至少一个 AI 模型。可以和刚才钳子助手配的用同一个，也可以用不同的。
  • 添加完模型后，点"测试"按钮，确认模型能正常使用。
  • 然后去左侧菜单的"仪表盘"或"服务管理"页面。
  • 找到 Gateway（可以理解为系统的"总开关"），点击"启动"按钮。
  • 等它变成绿色的"运行中"，就说明主引擎已经启动成功了。
过关标准： 模型测试通过，Gateway 显示"运行中"（绿色状态）。如果 Gateway 启动不了，直接去问钳子助手。
06
第一次对话 — 验证一切正常

发一条消息试试，看看系统是不是真的跑通了

到了激动人心的时刻！我们来验证一下前面所有的努力是否成功。

  • 在左侧菜单找到"实时聊天"，点进去。
  • 在输入框里输入一句话，比如："你好，介绍一下你自己"
  • 按回车发送。
  • 如果收到了回复 —— 恭喜你！整个系统已经跑通了！
  • 如果没有收到回复，不要慌：
    1. 回去检查 Gateway 是否还在"运行中"状态
    2. 或者直接问钳子助手："我在实时聊天里发消息没有回复，怎么办？"
过关标准： 在实时聊天里发消息，能收到 AI 的回复。到这一步，系统的核心功能已经完全跑通了。
07
连接聊天工具（可选）

让 AI 在飞书、Telegram 里也能回复你

如果你想让 AI 不只在 ProspectClaw 里能用，还能在飞书、Telegram 等聊天工具里也能回复，可以继续这一步。如果暂时不需要，可以直接跳过。

飞书 / Lark：

  • 需要在飞书开放平台创建一个"企业自建应用"，并开启机器人功能。
  • 拿到 App ID 和 App Secret（这两个是飞书给你的凭证），填到 ProspectClaw 的"消息渠道"页面。
  • 这一步操作比较复杂，如果你不知道怎么弄，可以让钳子助手一步步教你，或者让你的技术同事帮忙。

Telegram：

  • 在 Telegram 里搜索 @BotFather，发送 /newbot 来创建一个机器人。
  • 按提示给机器人起名字后，你会拿到一个 Token（一串字符），把它填到 ProspectClaw 的"消息渠道"页面。
  • Telegram 的接入相对简单，适合先试试看。

这一步涉及到第三方平台的操作，确实比前面几步更复杂一些。如果暂时用不上，完全可以以后再弄。

过关标准： 聊天工具里能收到机器人的回复。或者你选择了跳过这一步 —— 那也完全没问题。

飞书 / Lark

需要 App ID 和 App Secret。操作较复杂，建议让钳子助手或技术同事协助。

Telegram

只需要一个 Bot Token。操作简单，适合快速体验外部聊天渠道。

08
遇到问题怎么办

记住这三个求助方法，以后都不用慌

用软件难免会遇到问题，但你不需要自己搞定所有事情。这里是三个解决问题的方法，按优先级排序：

  • 第一选择：问钳子助手！在钳子助手里描述你的问题，比如"我点了启动但是 Gateway 还是红色的"，它会帮你一步步排查。
  • 第二选择：把这个网页发给任何 AI 助手。打开 ChatGPT、Claude 或者其他你在用的 AI，把这个页面的链接发给它，告诉它你的问题。它能根据这个页面的内容帮你分析。
  • 第三选择：查看进阶指南。进阶指南里有更详细的操作手册、排障清单和常见问题解答，适合想更深入了解的用户。

最重要的一条：不要一出问题就想重装！大多数问题都有简单的解决方法，重装反而可能丢失你之前的配置。先问 AI，让它帮你判断。

过关标准： 你知道遇到问题该找谁了 —— 第一时间找钳子助手，或者把这个页面发给 AI。
Quick FAQ

最常见的 4 个问题

🍎

Mac 提示"无法打开"或"已损坏"

这不是病毒，只是苹果的安全机制。解决方法：打开"系统设置" → "隐私与安全性" → 找到 ProspectClaw 的提示 → 点击"仍要打开"。详见第 2 步。

🔴

初始设置里有红色项

红色项表示需要安装某个工具。按页面上的提示操作即可。如果搞不定，配好钳子助手后直接问它："初始设置里 XX 是红色的，怎么办？"

🚧

Gateway 启动不了

最常见的原因是模型没配好或者端口被占用。不用自己排查，直接在钳子助手里说"Gateway 启动失败"，让它帮你诊断。

💬

实时聊天没收到回复

先检查：左侧菜单里的"服务管理"页面，Gateway 是否显示绿色的"运行中"？如果不是，先启动它。如果是绿色的还没回复，问钳子助手帮你查。

记住这个万能方法

任何时候遇到问题，你都可以把这个页面（或者截图）发给 AI 助手。无论是钳子助手、ChatGPT、Claude 还是其他 AI，它们都能根据这里的信息帮你解决问题。你不需要自己懂技术，让 AI 来帮你就好。
