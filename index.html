Getting Started

两条路径，快速上手

每一步都告诉你做什么、在哪做、做完算什么。下载即可开始。

📈

10 分钟跑通投资工作台

顺着这条链走，最快能把线索到审批的流程跑通。

  1. 1
    导入演示数据
    先让首页出现一组可浏览的任务和样例，后面的项目池、Deal 才有完整链路可看。
  2. 2
    在项目池录入第一条线索
    把项目名称、行业、负责人先放进项目池，先把线索进入系统。
  3. 3
    转成 Deal 并推进阶段
    把值得跟进的项目转成 Deal，体验阶段推进和待办记录。
  4. 4
    补齐 BP、初评或尽调材料
    把审批会用到的材料挂到文档库，让 Deal 和 SOP 围绕同一份资料工作。
  5. 5
    发起审批或执行 SOP
    材料齐套后推进到审批，让系统承担流程提醒和决策沉淀。
🦐

OpenClaw Agent 养成教程

按这 5 步把 OpenClaw 从"装好了"养到"第一只 Agent 真能用"。

  1. 1
    环境体检
    确认 Node.js、OpenClaw CLI 和配置文件是否齐全，先把底座检查清楚。
  2. 2
    给虾配脑子
    添加至少一个主模型，确保后面创建 Agent 时有可用的推理能力。
  3. 3
    开虾缸
    创建第一只 Agent，给它独立身份、模型和工作区。
  4. 4
    喂技能或装灵魂
    确认可用 Skill，或去 AI 助手页导入灵魂配置让 Agent 更像你的团队。
  5. 5
    让虾开口
    去实时聊天发第一条消息，确认 Agent 真正能对话。
Architecture

专业级技术底座
