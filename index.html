你的 Agent，你的规则

自定义 Agent

用自然语言描述角色、职责和边界，AI 帮你起草完整设定。从客服到分析师，从文档助手到代码审查员，Agent 的身份由你定义。

Skills 即能力

Skills 是 Agent 的手和脚。挂载文件操作、数据查询、API 调用等技能，让 Agent 不只会聊天，还能真正执行任务。

通过 MCP 协议接入任意工具和数据源。不需要写代码，只需要配置——数据库、REST API、文件系统、第三方服务，一切皆可连接。

Agent 不只在面板里工作——接入飞书、Telegram、Discord，让它在你的团队沟通工具中直接响应。

定时任务 + 心跳监控 + 事件触发，Agent 可以 7×24 自动运行。异常时自愈，需要时通知你，不需要时安静工作。

Architecture

专业级技术底座

Rust 后端保安全、Vanilla JS 保轻量、SQLite 保隐私、OpenClaw 保智能。

⚙

Tauri v2

Rust 后端 + 系统 WebView，安装包约 10MB，无需 Electron 臃肿运行时。跨平台支持 macOS、Windows、Linux。

⚡

Vanilla JS + Vite

零框架依赖，极致轻量。Vite 构建毫秒级热更新，原生 CSS 变量驱动主题系统。

SQLite 本地存储

自研 Agent 框架，支持 OpenAI、Anthropic、Deepseek 等多模型。4 种操作模式，8 种内置工具，MCP 扩展无上限。
